# voice-emotion-analysis
목소리 속 감정을 파악하는 프로그램

## Language, Development tools
Python 3, Pycharm(Anaconda, Virtual Machine)

## Main Dependencies
- PyAudio
- Librosa
- Keras
- Tensorflow

## Introduction
사용자의 목소리 속 감정 별 특성을 분석해내 현 기분(총 7가지)을 파악할 수 있는 프로그램

##  Predictive accuracy
60.87%
